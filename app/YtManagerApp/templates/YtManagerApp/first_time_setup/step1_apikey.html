Step 1: Set up YouTube API key (optional)

This program uses the YouTube API in order to obtain information about videos, channels and playlists. To access this API, YouTube requires users to register on their site and request an API key. YouTube uses this key to control access and limit the number of requests made to their platform, in order to prevent abuses.

To use this program, it is recommended, but not required, that you create an API key for your own account. While a key is already provided the developer, there is a chance that the quota limits will be reached, which will prevent this program from reaching YouTube. Follow the steps below, or press Skip to use the provided key.

How to obtain a key

  1. Visit https://developers.google.com/, log in or create an account, if necessary.
  2. Go to , and click on the Create project button.
  3. Give a name to the project, and then click on Create. Wait for a few seconds, until Google finishes creating the project.
  4. Make sure the newly created project is selected in the top bar and then, in the left sidebar, go to APIs & Services - Library.
  5. Find and click on the YouTube Data API v3 from the list.
  6. Click Enable to enable the YouTube API for your account.
  7. In the navigation sidebar, go to the Credentials page.
  8. Click on Create credentials.
  9. Fill the requested information; we will need access to the YouTube Data v3, we will be calling the API from a Web server, and we will access the Public data. After filling the information, click on the What credentials do I need? button.
  10. Copy the created key in the box below, and then hit Done.
